# **App Name**: LeafPulse Dashboard

## Core Features:

- Data Storage: Store IoT device logs and leaf health data in Firestore.
- Real-time Chart Display: Visualize real-time leaf health data using interactive charts.
- Disease Alert System: Provide alerts based on detected abnormalities using ML-based reasoning as a tool.
- IoT Device Logs: Display historical IoT device logs for analysis.
- Manual Controls: Implement buttons for manual device control.

## Style Guidelines:

- Primary color: Forest green (#388E3C) to reflect plant health.
- Background color: Light beige (#F5F5DC) for a natural feel.
- Accent color: Earthy brown (#A1887F) for highlighting key data.
- Body font: 'PT Sans', sans-serif, providing modern readability.
- Headline font: 'Space Grotesk', sans-serif, offering a techy look, paired with PT Sans for longer text.
- Use minimalistic icons representing leaf health and IoT devices.
- Clean and organized layout for easy navigation.
- Subtle animations to indicate real-time data updates.
